Preferences - Main View

This dialog is accessed by clicking on the Main View tab in the Preferences Dialog.

Map contain options that alter the way the main view ( map or list ) of the world is displayed

Map Size: This lets you display the map in its actual size ( the most accurate ) or resize the map so that it stretches to fit the size of the window. Note: The program will use more resources the larger the map is.

Night Mask: This is a color shading that is drawn over the map to give a quick visual que which part of the world is in daytime, and which is at night. You can can set which colors to mask with the map to respresent night and twilight. This option may also be turned off. Note: This is only an approximation

List View Mask: This is a color shading that is drawn over the the map when the application is in list view mode

The 2 vertial lines on the map represent Midday and Midnight.
