Preferences - Locations

This dialog is accessed by clicking on the Locations tab in the Preferences Dialog.

Locations lets you alter the way Locations are shown on the map

Background Type: This lets you set how you want the backgroun color for the locations displayed. You can select Translucent, Solid or None.

Use Template rule on selected Location: When this option is enabled, then the selected location, or the one in focus will be displayed occording to the template rules ( see Tamplate tab ). Otherwise the selected location will be displayed in the same manner as the other locations.

Show all locations at once: Enabling this option shows all locations in the display list on the map. Disabling this option only shows the selected location, while the other locations will be represented by a red dot.

The selected locations position on the map is either displayed as a globe or a face depending on whether you selected this display item to be a location ( globe is displayed ) or a person ( face is displayed )
